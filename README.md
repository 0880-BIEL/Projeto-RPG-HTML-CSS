# ⚔️ Projeto Ficha de RPG

Um projeto simples e prático desenvolvido para criar uma interface interativa de ficha de personagem de RPG de mesa. O objetivo principal foi treinar conceitos de estruturação e estilização web.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Para a estruturação dos campos da ficha (atributos, dados do jogador, vida, defesa, etc.).
* **CSS3:** Para a estilização visual, organização das seções em blocos coloridos e posicionamento dos elementos na tela.
